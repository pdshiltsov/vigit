## About

**vigit** is a lightweight terminal user interface (TUI) project focused on simplifying everyday Git workflows.

The project is currently in active development, so the architecture, feature set, and interface may change over time.

## Status

Work in progress

At the moment, there are no stable or finalized features yet.
